你好，欢迎来到 npm 冒险训练营！  我将要求你做各种与 npm 相关的事情，这样你就可以很容易的开始
用它了。

一些有用命令列表: 

npm help ............ 可获得 npm 方面的帮助
how-to-npm print .... 重新显示当前的练习
how-to-npm verify ... 验证你是否完成了一项练习 
how-to npm solution . 展示当前练习的答案

首先，请确认你是否安装了最新版的npm。 

如果确认成功，请运行 “$ADVENTURE_COMMAND verify” 。 

（在此训练营中唯一需要网络访问的部分。
如果你处于一个网络不通的状态，你可以运行 “$ADVENTURE_COMMAND verify skip” 跳过。 ）
